## 解决方案

### 统一模板
```
某商家在[平台]销售[商品名称]。

商品价格信息：
- [价格标注描述]

经调查核实：
- [事实1]
- [事实2]
- [事实3]

请分析该经营行为是否存在价格问题，并给出法律依据和结论。
```

### 匿名化规则
| 原始内容 | 替换为 |
|---------|--------|
| 具体店铺名称 | 某商家 |
| 具体公司名称 | 某公司 |
| 具体地址 | [删除] |
| 身份证号、信用代码 | [删除] |
| 监管部门名称 | [删除] |

### 禁止词
"违规"、"违法"、"合规"、"查处"、"处罚"、"虚构原价"、"虚假折扣"、"价格误导"、"要素缺失"
